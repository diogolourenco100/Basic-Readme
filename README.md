
# Caeser Cipher  
## Run Locally  

Clone the project  

~~~bash  
git clone https://github.com/USER/REPOSITORY
~~~

Go to the project directory  

~~~bash  
cd ScriptRepo
~~~

Install dependencies  

~~~bash  
pip install -r requirements.txt
~~~

# Start the script

Linux
~~~bash  
python3 script.py
~~~
Windows
```bash
py script.py
```
